## Renuka Guest House – Hampi

This is a small marketing site for **Renuka Guest House** in Hampi, built with **Next.js App Router**.

### Tech stack
- **Framework**: Next.js (App Router, `app/` directory)
- **Styling**: Tailwind CSS (via `globals.css` and utility classes)
- **Fonts**: Google Fonts (`Poppins`, `Inter`) using `next/font`

### Getting started
```bash
pnpm install   # or npm install / yarn
pnpm dev       # or npm run dev / yarn dev
```

Then open `http://localhost:3000` in your browser.

### Important files
- `app/layout.js` – Root layout and SEO `metadata`
- `app/page.js` – Home page content
- `app/globals.css` – Global styles and Tailwind base

### Deployment
You can deploy this app easily to platforms that support Next.js (for example, Vercel). Make sure environment variables (if you add any) are configured in the hosting dashboard.
